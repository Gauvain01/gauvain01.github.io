# gauvain01.github.io

Personal CV website for GitHub Pages.

## Structure

- `index.html` — structured professional CV website (HTML) based on the cleaned CV content.
- `assets/cv.pdf` — cleaned downloadable CV file.
- `_config.yml` — Jekyll metadata for GitHub Pages.

## Update the CV

Replace `assets/cv.pdf` with your newest version and push to `main`.

## Deploy on GitHub Pages

1. Push this repository to GitHub.
2. In GitHub, open **Settings → Pages**.
3. Set **Build and deployment** to **Deploy from a branch**.
4. Select branch `main` and folder `/ (root)`.
5. Save and wait for GitHub Pages to publish.
